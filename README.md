# 💜 AcolheEdu

O AcolheEdu é uma plataforma web desenvolvida para promover o bem-estar emocional de estudantes dentro do ambiente escolar.

O sistema oferece ferramentas de acolhimento, comunicação e apoio psicológico, permitindo que alunos encontrem suporte de forma segura, acessível e acolhedora.

## 🎯 Objetivo

O projeto tem como objetivo aproximar estudantes da equipe de apoio escolar, fornecendo recursos para:

- Registro emocional diário
- Avaliação de bem-estar
- Conversa anônima
- Apoio psicológico
- Canal de denúncias
- Atendimento à mulher

---

## ✨ Funcionalidades

###  Diário Emocional
Permite que o estudante registre emoções, sentimentos e experiências do dia.

###  Bem-estar
Questionários rápidos para acompanhamento do estado emocional do aluno.

###  Conversa Anônima
Canal seguro para comunicação sem exposição da identidade do estudante.

###  Apoio Psicológico
Área para solicitação e agendamento de atendimento psicológico.

###  Denúncias
Canal para denúncias de bullying, assédio e violência emocional.

###  Atendimento à Mulher
Espaço dedicado ao acolhimento e orientação para estudantes mulheres.

###  Perfil
Gerenciamento de informações do usuário, incluindo:

- Alteração de nome
- Alteração de senha
- Alteração de foto de perfil

###  Configurações
Preferências da plataforma, notificações e configurações gerais.
