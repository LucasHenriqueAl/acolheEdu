# acolheEdu
 O AcolheEdu é uma plataforma digital criada para aproximar estudantes da psicologia escolar de maneira rápida, segura e acolhedora. A solução funciona através de aplicativo e site, onde o aluno poderá conversar anonimamente com a psicóloga, agendar atendimentos, denunciar casos de bullying e violência emocional.
